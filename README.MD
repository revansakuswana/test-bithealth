## Instalasi

```bash
git clone https://github.com/revansakuswana/test-bithealth.git
```

### Install dependencies (npm or yarn)

```bash
cd frontend/cd backend
npm install
```

### Start the development server

```bash
npm run dev (backend)
npm run dev (frontend)
```
